# Eventos
### Multi Event Management Platform

## Description

Eventos is a peer-to-peer event management platform where users can create, manage, and participate in multiple events. Admins can effortlessly organize their events, while participants can join and keep track of events they love. We make event management simple, interactive and organized—so everyone can focus on enjoying the experience, not the logistics!

## Features-

### For Admin
* Create, edit, and manage multiple events effortlessly.
* View a dashboard of all events you’ve organized.
* Track participant registrations and event details.
### For Participant
* Browse and join events seamlessly.
* Access a personalized list of events you’ve participated in.
* Get notifications and updates for upcoming events.
### General
* Role-based authentication: Admins and Participants have tailored experiences.
* Interactive and responsive design built with the MERN stack.
* Efficient event tracking and management in one platform.
* Easy-to-use interface with smooth navigation.


### Tech Stack

| Layer                    | Tech                 |
| Frontend                 | React JS             |
| Database                 | Mongo DB Atlas       |
| Backend                  | Node JS, Express JS  |

## Usage Flow

| Step | Action                                                                   |
| 1    | Home Page                                                                |
| 2    | User signs up/Existing user logs in                                      |
| 3    | Admin can view dashboard of his organized events (Manage Events)         |
| 4    | Admin can create a new event (Create Event)                              |
| 5    | Participant can view the events in which he has participated (My Events) |
| 6    | Participant can participate from a bucket of events (Explore Events)     |

| 3    | models/backend                    |
| 4    | routes/backend                    |
| 5    | utils/backend                     |

## Design

* Seamless multi-event management

* Easy event creation, participation, and tracking
* Interactive, user-friendly interface
* Centralized dashboard for events and registrations
* Encourages organized and engaging event experiences
* Built with a modern MERN stack for responsiveness and scalability
